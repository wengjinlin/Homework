# FTP程序说明

> - 用Python3.0及以上解释器运行，运行环境保证编码为UTF-8
> - 程序服务器端入口为：FTP\server\bin\server.py
> - 程序客户端入口为：FTP\client\bin\client.py
> - 服务器端IP和端口设置在FTP\server\conf\config.py文件中，客户端运行后手动输入ip地址和端口号
> - 服务器端可以接收多个客户端的连接，但同一时间只能有一个通讯
> - 程序在windows系统运行正常，客户端必须安全退出，不能强制断线，否则服务端会崩溃
> - 可以在linux系统上运行，实现与windows的通讯，但是目前有一个bug，上传和下载文件的时候，无法自动完成退出，卡在传输界面，但是文件传输是完成的，估计是最后一个包的接收问题，暂时无法解决
> - 传输文件大小没有限制，自测成功传输1G的MP4文件
> - 系统有四个功能
> - 登录功能：
>   - 1.用户手动登录
>   - 2.其他各功能均判断是否登录，需登陆后进行后续操作
> - 查看功能：
>   - 1.查看用户自己的目录文件
> - 上传功能：
>   - 1.向用户自己的目录上传文件，可传输任意文件，大小不限
>   - 2.不能传输文件夹，以及建立二级目录，只能直接传输文件
> - 下载功能：
>   - 1.显示当前用户的目录文件
>   - 2.输入文件名后进行下载，默认下载目录为客户端下的download文件夹
> - 具体测试用例见test.md
